üí° Recommendation

Option 2 (Vertical Side Stepper) provides the best balance of usability and space efficiency for the JML wizards. It allows users to see all steps at a glance, provides contextual descriptions, and keeps the panel width reasonable while maintaining enough space for form fields.

For mobile/tablet, consider falling back to Option 3 (Pill Tabs) which adapts better to narrow viewports.
